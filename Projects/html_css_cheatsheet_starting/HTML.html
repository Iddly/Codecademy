Syntax to write an HTML element

Description
Opening Tag Every element has an opening tag with the name of the element at its start.
Attribute and its value (optional) Attribute are like options of an element. Attribute have value.
Closing Tag A closing tag has the name of the element with a forward slash "/" before it.
Children (optional) Between the opening and closing tags are the children of the element. This can be more elements or just plain text.
Description
Self-Closing Tag Some elements that do not have children do not need a closing tag. In this case the forward-slash "/" is added to the element's opening tag to denote a self-closing tag.
